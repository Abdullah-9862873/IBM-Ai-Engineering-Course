# Quiz Answers Review

## Question 1: Best describes machine learning?

**Options:**
1. ML enables computers to automatically make decisions without data or explicit instructions
2. ML enables computers to detect anomalies and find patterns to assign data into similar groups
3. ML teaches computers to learn under supervision how to make predictions from labeled data
4. ML teaches computers to learn from data, identify patterns, and make decisions without explicit human instructions

**Answer: 4** ✅

**Reason:** This is the complete definition of ML from the course. Options 1-3 are incorrect because:
- Option 1: ML needs DATA
- Option 2: Only describes unsupervised learning (clustering/anomaly detection)
- Option 3: Only describes supervised learning

## Question 2: ML Engineer more likely to perform than Data Scientist?

**Options:**
1. Creating informative plots for illustrating key insights (Data Scientist)
2. Research the latest trends in data analytics (Data Scientist)
3. Build a data pipeline for deploying a machine learning model
4. Investigate causal relationships by developing hypotheses (Data Scientist)

**Answer: 3** ✅

**Reason:** ML Engineers focus on **deployment** and building pipelines. Data Scientists focus on analysis, insights, and hypotheses.
