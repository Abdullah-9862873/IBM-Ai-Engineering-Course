# Quiz: Keras and Deep Learning Libraries

## Question 1
Which of the following libraries is particularly suited for running machine learning algorithms on GPUs?

**Answer: PyTorch**

---

## Question 2
Which of the following statements is correct?

**Answer:** Keras is a high-level API that facilitates fast development and quick prototyping of deep learning models.

---

## Question 3
What are the two model classes in the Keras library? Select all that apply.

**Answer:**
- The model class used with the functional API ✓
- Sequential model ✓

---

## Question 4
When adding a 'dense' layer to the first hidden layer in a regression model, what three things do we specify?

**Answer:** The number of neurons in each layer, the activation function, and the number of columns in our data set.

---

## Question 5
When building classification models with Keras, what do we need to do to the target column before we can use it?

**Answer: Transform it into an array of binary values**

**Answer: The Sequential model**

## Question 4 (Practice Quiz)
You're building a Keras model to classify images into 10 different categories. Which activation function should you use in the output layer?

**Answer: Softmax activation function**

---

## Question 5 (Practice Quiz)
For regression problems involving continuous target variables, which output layer activation function provides the most flexibility?

**Answer: Linear activation function**

---

## Question 6 (Practice Quiz)
For a neural network to predict categorical outputs, which loss function optimally measures prediction accuracy?

**Answer: Categorical Cross-entropy**

---

## Question 7 (Practice Quiz)
You've trained a Keras model to predict stock prices and want to assess its performance. Which evaluation metric is most appropriate?

**Answer: Mean Squared Error (MSE)**

---

## Question 8 (Practice Quiz)
When implementing a Keras regression model, which compilation parameter should be set to optimize for continuous outcome prediction?

**Answer:** loss='mean_squared_error', optimizer='adam', metrics=['mean_absolute_error']

---

## Question 9 (Practice Quiz)
For evaluating classification model performance in a lab setting, which metric offers the most comprehensive assessment of model effectiveness?

**Answer: Accuracy score**

---

## Question 10 (Practice Quiz)
A student new to deep learning chooses Keras, TensorFlow, and PyTorch. What makes Keras the most beginner-friendly option?

**Answer:** Keras offers intuitive syntax and simplified model-building processes.
